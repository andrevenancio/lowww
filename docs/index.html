    lowww engine

    lowww is a WebGL 2.0 3D engine. This is an experimental project focused in the advantages of WebGL 2.0 and GLSL ES 3.0. It falls back to WebGL for devices that don't yet support WebGL 2.0, but this functionality will be removed as soon as WebGL 2.0 becomes standard.

    All functionality is separated per package, each one in its own npm module.

    Packages

    name package description
    controls npmjs Camera control utilities.
    core npmjs Handles all core functionality of the engine.
    geometries npmjs A collection of platonic solids and other geometries with positions, indices, uvs and normals.
    postprocessing npmjs Postprocessing effects to use with build in Composer and Pass.

    Development

    run npm run dev:all to run all dev tasks concurrently with an http-server. To run a specific dev task you will need to enable the server manually like npm run dev:geometries & npm run server

    Production

    run npm build:all to run all the build tasks and generate documentation.
